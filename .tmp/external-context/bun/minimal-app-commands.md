## script runner

- `bun run <script>` executes `package.json` scripts
- `bun <script>` is a shorthand, but `bun run` is clearer and avoids name conflicts
- `bun ./index.ts` runs a TS/JS entry file directly

## test runner

- `bun test` runs the project test suite

## useful minimal commands

- `bun install`
- `bun run dev`
- `bun run build`
- `bun test`
