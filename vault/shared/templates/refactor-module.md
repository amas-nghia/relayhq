## Objective

Improve the target module structure without changing user-visible behaviour.

## Acceptance Criteria

- Behaviour stays unchanged
- Complexity is reduced
- Tests still pass

## Context Files

- web/src/
- app/server/

## Constraints

- Prefer small, reviewable refactors
